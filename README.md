# Sales Data Analysis Project

This project analyzes 12,000+ sales records using Python and Pandas.

## Objectives
- Revenue analysis
- Profit analysis
- City performance evaluation
- Product performance insights
- Monthly sales trend analysis

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
Synthetic sales dataset with 12k+ transaction records.

## Key Insights Generated
- Identified top revenue generating cities
- Evaluated product profitability
- Built monthly sales trend analysis
- Created automated KPI reporting
